Nutrimed is a responsive web application that generates personalized diet plans based on user health information and preferences.

Features
  User authentication
  Health information and preference collection
  Customized 4-week nutrition plan generation using GROQ AI
  Responsive design for various devices
  Modern, utility-first styling with Tailwind CSS

Usage
  Register an account or log in
  Complete the health information and preference questionnaire
  Receive your personalized 4-week nutrition plan

Tech Stack
  Frontend: React.js
  Backend: Node.js, Express.js
  Database: MongoDB
  AI Diet Plan Generation: GROQ AI API
  CSS Framework: Tailwind CSS
