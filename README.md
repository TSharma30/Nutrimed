Nutrimed is a responsive web application that generates personalized diet plans based on user health information and preferences.

Features
  1. User authentication
  2. Health information and preference collection
  3. Customized 4-week nutrition plan generation using GROQ AI
  4. Responsive design for various devices
  5. Modern, utility-first styling with Tailwind CSS

Usage
  1. Register an account or log in
  2. Complete the health information and preference questionnaire
  3. Receive your personalized 4-week nutrition plan

Tech Stack
  1. Frontend: React.js
  2. Backend: Node.js, Express.js
  3. Database: MongoDB
  4. AI Diet Plan Generation: GROQ AI API
  5. CSS Framework: Tailwind CSS
